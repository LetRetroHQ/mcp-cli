# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in @letretro/mcp, please report it privately.

**Do not open a public GitHub issue.**

Send details to **team@letretro.com** with the subject "[Security] @letretro/mcp vulnerability".

We'll acknowledge receipt within 48 hours and work on a fix. We'll keep you updated
on progress and credit you when the fix is released.

## Scope

- The `@letretro/mcp` npm package (src/)
- The installer scripts (config file handling)
- The proxy server (API key handling in server.ts)

## Out of scope

- The hosted MCP server at `mcp.letretro.com`
- The LetRetro web application
